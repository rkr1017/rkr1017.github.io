Stroop Dataset Analysis

Variables

Independent variable: word condition, congruent and incongruent. Dependent variable: the time it takes to name the ink colors in equally-sized lists.

Hypotheses

Information about a word can interfere with the information about a color of this word as seen on screen. As a result, when congruent, colors seem to be named faster than when incongruent. We are trained to read words and when the meaning of the word does not correspond to the color we spend additional time deciding which is correct: color or word naming it.
We are trying to find, based on available small samples of congruent and incongruent response times, if those response times are different at population level. We can see the difference in samples and it seems like response times for incongruent are larger than for congruent, the question remains if this is likely for the whole population?
The null hypothesis is that here is no difference in population means between times for congruent and incongruent. The alternative hypothesis can be framed in two ways.
1. The incongruent response time is different from the congruent. Alternative hypothesis: true difference in population means is not equal to zero. In this case we need to perform a two-tailed two sample paired t test.
2. Looking at the data it seems like the values of incongruent response time are always higher than of congruent for each individual. If we want to test for a greater incongruent response times then the alternative hypothesis is: true difference in population means is greater than zero. In that case we have to perform a one-tailed two sample paired t test.
The two samples are not independed of each other. Same subjects perform both tests. Because of that samples are not independent and paired test should be used.

Statistical test

Both one-tailed and two-tailed paired t test will be performed. The two samples are not independed of each other. Same subjects perform both tests. Because of that samples are not independent and paired test will be used.

Results

Histograms of both groups congruent and incongruent probably indicate normal distribution, but it is had to tell for sure due to a small sample size. Boxplots reveal a couple of outliers in incongruent sample. In this study a paired test is implemented, so outliers are better revealed by plotting of differences in values between two groups. Boxplot of differences reveals one outlier, which agrees with a q-q plot as well. If outlier is removed, the boxplot and q-q plot show better agreement with normality, which is also confirmed by Shapiro-Wilk test. Shapiro-Wilk test results are in favor of normal distribution when the outlier is dropped and indicate departure from normality before the outlier removal. The outlier was removed from both samples to create a new dataframe for further analysis.

Tests were performed on a modified dataset. As a result of the test we can conclude that the incongruent condition does increase time to respond with the naming of the color.

There are various theories with regard to observed effect [1]. An automaticity model, a speed pf processing model and a paralell distributed processing model. According to the automaticity, reading wors is automatic and happens fast, but naming of clors is not thus interference. A speed of processing model, which I like better, states that reading is faster, so when color information is processed later it creates confusion. According to a third theory both processes happen in parallell and reading is a stronger one which interferes with the color response. I would think we ate trained to read if we see a word no matter what color or style it is and understand the meaning first which interferes with the information of a incongruent color and takes time to sort it out. At the same time all theories may overlap to some extent.
